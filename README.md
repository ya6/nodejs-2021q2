## Установка
- Рекомендую создать отдельную папку например _tests
  
  mkdir _tests
  cd _tests
  git clone https://github.com/ya6/nodejs-2021q2.git
  cd  nodejs-2021q2
  git checkout cli-tool
  npm install
  node index <+ваши проверки>
- например:
 node index -a encode -s 7 -i input.txt
